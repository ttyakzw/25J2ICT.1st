💡 CSSって何？

CSSの正式名称

CSS = Cascading Style Sheets

  • Style： スタイル（見た目）
  • Sheets： シート（設定をまとめたもの）

HTMLとCSSの役割分担

📄 HTML = 構造
「これは見出し」「これは段落」というページの骨組みを作る
🎨 CSS = 見た目
色、大きさ、配置など、デザインを整える

✏️ CSSの基本的な書き方

基本の形

セレクタ { プロパティ: 値; }

具体例

h1 { color: blue; font-size: 40px; }

各部分の説明

セレクタ（h1）
どのタグに適用するか（h1, h2, p など）
プロパティ（color）
何を変えるか（色、大きさ、位置など）
値（blue）
どんな値にするか（blue, 20px など）
⚠️ 重要なルール
• プロパティと値の間は「:（コロン）」
• 各行の最後に「;（セミコロン）」を必ず付ける
• { } で囲む（波括弧・中括弧）
• 全て半角英数字で書く

📐 レイアウトの基本

🌟 レイアウトとは？
要素の配置や並び方を整えることです。見やすく、使いやすいページを作るために大切です。

width と height - 幅と高さ

要素の大きさを指定
width = 幅、height = 高さを指定します。
div { width: 300px; height: 200px; background-color: lightblue; }
表示結果
幅300px、高さ200pxのボックス

max-width - 最大幅

要素の最大幅を制限
画面サイズに応じて自動調整される幅を指定します。
img { max-width: 100%; height: auto; }
💡 重要： 画像が画面からはみ出さないようにするテクニックです

display - 表示方法

要素の並び方を変える
block = 縦に並ぶ、inline = 横に並ぶ、inline-block = 横に並ぶけど幅や高さを指定できる
p { display: inline-block; width: 200px; background-color: lightblue; margin: 5px; }

📱 レスポンシブデザインとは？

🌟 レスポンシブデザインとは？
スマホ、タブレット、パソコンなど、どんな画面サイズでも見やすく表示されるデザインのことです。

なぜ大切？

  • 今は多くの人がスマホでWebページを見ている
  • 画面サイズに合わせて自動調整されると便利
  • プロのWebサイトは全てレスポンシブ対応している

基本テクニック

1. 画像を画面に合わせる
img { max-width: 100%; height: auto; }
画像が画面からはみ出さず、画面幅に合わせて自動で縮小されます。
2. コンテンツの幅を制限
body { max-width: 800px; margin: 0 auto; }
大きな画面でも読みやすい幅に制限し、中央に配置します。
3. %（パーセント）を使う
div { width: 80%; margin: 0 auto; }
画面幅の80%の大きさになるので、どんな画面でも自動調整されます。
🎯 今日覚えるレスポンシブの基本
• 画像に max-width: 100%; を付ける
• 固定のpxより、%を使うと柔軟になる
• スマホでも見やすいデザインを意識する

発展課題1：クラスセレクタを使おう

特定の要素だけにスタイルを適用
今まではh1やpなど、全ての要素に同じスタイルが適用されました。クラスを使うと、特定の要素だけにスタイルを適用できます。

HTML側：

<p class="important">これは重要な段落です</p> <p>これは普通の段落です</p>

CSS側：

.important { color: red; font-weight: bold; background-color: lightyellow; padding: 10px; border-left: 4px solid red; }
表示結果

これは重要な段落です

これは普通の段落です

💡 ポイント： CSS側では .（ドット）を付けてクラス名を書きます

発展課題2：ホバーエフェクトを追加しよう

マウスを乗せたときに色が変わる
:hover を使うと、マウスを乗せたときだけスタイルを変えられます。
a { color: blue; text-decoration: none; } a:hover { color: red; text-decoration: underline; }
💡 試してみよう： リンクにホバーエフェクトを付けると、クリックできることが分かりやすくなります

発展課題3：グラデーション背景を作ろう

2色以上の色が混ざった背景
linear-gradient を使うと、色が徐々に変わる背景が作れます。
body { background: linear-gradient(to right, lightblue, lightpink); } h1 { background: linear-gradient(to bottom, #667eea, #764ba2); color: white; padding: 20px; }
表示結果

グラデーション見出し

💡 方向： to right（左→右）、to bottom（上→下）、to top（下→上）、to left（右→左）

発展課題4：影を付けよう

box-shadow で立体感を出す
影を付けると、要素が浮き上がって見えます。
h1 { background-color: white; padding: 20px; box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2); }
表示結果

影付きの見出し

💡 書き方： box-shadow: 横の位置 縦の位置 ぼかし 色;

発展課題5：Flexboxでレイアウトを整えよう

要素を横並びにする
Flexboxを使うと、要素を簡単に横並びにできます。

HTML側：

<div class="container"> <div class="box">ボックス1</div> <div class="box">ボックス2</div> <div class="box">ボックス3</div> </div>

CSS側：

.container { display: flex; gap: 10px; } .box { background-color: lightblue; padding: 20px; border-radius: 10px; flex: 1; }
表示結果
ボックス1
ボックス2
ボックス3
💡 Flexboxの便利なプロパティ：
• justify-content: center; （中央揃え）
• align-items: center; （縦方向の中央揃え）
• flex-direction: column; （縦並び）

発展課題6：アニメーションを追加しよう

動きのある要素を作る
transitionを使うと、スタイルの変化がスムーズになります。
button { background-color: blue; color: white; padding: 10px 20px; border: none; border-radius: 5px; transition: all 0.3s; } button:hover { background-color: darkblue; transform: scale(1.1); }
💡 transition： all 0.3s は「全てのプロパティを0.3秒かけて変化させる」という意味です

📚 HTMLをすでに知っている人向けの補足情報

CSSの詳細セレクタ

セレクタ 意味 例
.クラス名 特定のクラスを持つ要素 .important { color: red; }
#id名 特定のIDを持つ要素 #header { font-size: 30px; }
要素:hover マウスを乗せたとき a:hover { color: red; }
要素 要素 子孫要素（入れ子） div p { color: blue; }
要素 > 要素 直接の子要素 ul > li { font-weight: bold; }

よく使う単位

px, %, em, rem の違い
  • px（ピクセル）： 絶対的な大きさ。20px = 画面上の20ドット
  • %（パーセント）： 親要素に対する割合。width: 50% = 親の半分
  • em： 親要素のフォントサイズに対する倍率。2em = 親の2倍
  • rem： ルート要素（html）のフォントサイズに対する倍率
💡 おすすめ： レスポンシブデザインには % や rem が便利です

カラーコードの詳細

16進数カラーコード
#RRGGBB の形式で、RR=赤、GG=緑、BB=青の量を16進数で指定します。
  • #FF0000 = 赤（R=最大、G=0、B=0）
  • #00FF00 = 緑（R=0、G=最大、B=0）
  • #0000FF = 青（R=0、G=0、B=最大）
  • #FFFFFF = 白（全て最大）
  • #000000 = 黒（全て0）
💡 便利ツール： 「color picker」で検索すると、視覚的に色を選んでコードを取得できます

CSSの優先順位

同じ要素に複数のスタイルが指定された場合
優先順位（高い順）：
  1. インラインスタイル（HTML内に直接書く style="..."）
  2. IDセレクタ（#id）
  3. クラスセレクタ（.class）
  4. 要素セレクタ（h1, p など）
💡 ポイント： より具体的なセレクタほど優先されます

開発者ツールの使い方

ブラウザの開発者ツールでCSSを確認・編集

  1. CodePenのプレビュー画面で右クリック→「検証」または「要素を調査」
  2. 要素にマウスを乗せると、該当部分がハイライトされる
  3. 右側のパネルでCSSを確認・一時的に編集できる
  4. どのCSSが適用されているか、どれが上書きされているかが分かる
💡 プロの技： 開発者ツールで試してから、CodePenに反映させると効率的です

⚠️ よくあるエラーと解決法

❌ CSSが効かない
原因： セミコロン（;）を忘れている
解決法： 各行の最後に ; を付ける
❌ 色が変わらない
原因： 全角で書いている、またはスペルミス
解決法： 半角英数字で書き直す、color のスペルを確認
❌ デザインが崩れた
原因： 波括弧 { } の閉じ忘れ
解決法： { と } のペアを確認する
❌ 背景色が変わらない
原因： background-color のスペルミス
解決法： ハイフン（-）を忘れずに、background-color と書く

🎯 成功のコツ

  • セミコロン（;）を忘れずに付ける
  • 波括弧 { } はペアで使う
  • 色の組み合わせを工夫する（薄い背景色 + 濃い文字色）
  • 何度も試して、自分好みのデザインを探す
  • 困ったらこの資料を見返そう
  • 失敗を恐れずに色々試してみよう！
